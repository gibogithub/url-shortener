# URL Shortener Microservice

A Next.js implementation of freeCodeCamp's URL Shortener Microservice project.

## Features
- Shorten URLs via POST `/api/shorturl`
- Redirect via GET `/api/shorturl/:id`
- DNS-based URL validation
- In-memory storage
